# Random Password Generator

A simple random password generator built using HTML, CSS and some JavaScript.

## Features:
- Generate random passwords
- 12-character secure password generation
- Copy password to clipboard
- Simple and clean user interface

## Technologies Used:
- HTML
- CSS
- JavaScript
  
## How It Works:
- Click the Generate Password button and a random 12-character password will appear.
- Click the Copy button to copy the password and paste the password wherever you want.
